Deploy Detector on Jetson AGX Orin

This page describes a deployment tutorial in a wiki style.

Hardware Requirements

  • NVIDIA Jetson AGX Orin 32GB
  • USB camera or CSI camera
  • At least 20GB free storage

Software Requirements

  • Ubuntu 22.04
  • JetPack 6.2
  • CUDA 12.2
  • Python >= 3.10
  • PyTorch 2.4
  • TensorRT 10
  • ONNX Runtime 1.19
