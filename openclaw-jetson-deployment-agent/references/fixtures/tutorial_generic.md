# Generic Tutorial: Deploy Streaming App on Jetson Xavier NX

Requirements:

- NVIDIA Jetson Xavier NX
- JetPack 5.1.2
- CUDA 11.4
- Python >= 3.8
- PyTorch 2.1
- ONNX Runtime 1.17
- 8GB RAM minimum
- 10GB free disk

Steps:

1. Install project dependencies.
2. Run the app from a Python virtual environment.
